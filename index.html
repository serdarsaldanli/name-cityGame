Name&City Game

Game Rules

Computer gives a letter from the alphabet. The name, city, animal, plant, country which starting with the chosen letter, is written under the relevant item on the paper divided into items with longitudinal lines. The first player to complete these names takes 1 minute, the other players must finish within 1 minute. If 2 people write the same thing, it gets 5 points, if it is different, 10 points, if it is blank, it gets 0 points. The person with the most points at the end of the game wins.
